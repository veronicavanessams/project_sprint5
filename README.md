# Projeto Sprint 5 – Dashboard de Veículos 

Este projeto faz parte da Sprint 5 do curso de Análise de Dados da TripleTen.

##  Descrição
O objetivo é criar um *Dashboard interativo* usando *Python + Streamlit*, que exibe a distribuição dos hodômetros e outras análises do dataset vehicles.csv.

##  Tecnologias utilizadas
- Python  
- Pandas  
- Plotly Express  
- Streamlit
